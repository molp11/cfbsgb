An Interactive Tool for Commercial Fragment Library Comparison

- Use the dropdown menus to select descriptors to plot on the heatmap and calculate distribution statistics in the table.
- Use the "mean", "median", "mode", and "variance" buttons to sort the libraries in the heatmap by the current descriptor.
- Hover over the heatmap to view x and y axis values.
- Use the tools to the right of the scatter plot to pan, zoom, refresh and save the active plots.
* References for selected descriptors are located at the bottom of the page.
** Chemical descriptors were calculated using the RDKit python package (http://www.rdkit.org/).
